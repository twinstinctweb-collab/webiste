# FitSpace Pro - Gym Website Template

A premium spatial UI gym website template built with HTML, CSS, JavaScript, GSAP animations, glassmorphism, 3D hover cards, dock navigation, and editable client data.

## Files
- `index.html` - Main homepage
- `contact.html` - Contact page with Google Maps embed
- `style.css` - Full responsive styling
- `script.js` - Animations, config injection, cursor, 3D tilt
- `config.js` - Edit this file for each client

## How to Customize
Open `config.js` and edit:

```js
const gymConfig = {
  gymName: "Your Gym Name",
  brandName: "YOURGYM",
  phone: "9876543210",
  whatsapp: "919876543210",
  trainerName: "Trainer Name",
  monthlyPrice: "1200",
  offerPrice: "3000",
  address: "Your gym address here",
  mapQuery: "Your gym address here",
  designer: "Your Brand"
};
```

## How to Run
Just open `index.html` in a browser.

For best results, use a local server like VS Code Live Server.
