# Premium Cafe Template - Twinstinct Digital

A reusable premium cafe landing page template.

## Customize Quickly

1. Change cafe name in `index.html`.
2. Replace phone number in WhatsApp and call links.
3. Change location/map iframe.
4. Update menu items and prices.
5. Replace gallery placeholders with real images.
6. Change colors in `css/style.css` under `:root`.

## Sections Included

- Hero
- About
- Menu Preview
- Gallery
- Reviews
- Reservation CTA
- Contact
- Google Maps
- WhatsApp Button

Built using HTML, CSS, and JavaScript.
