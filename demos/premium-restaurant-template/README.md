# Premium Restaurant Template

A reusable premium restaurant landing page template for Twinstinct Digital.

## Sections
- Hero
- Stats
- About
- Signature Menu
- Special Offer
- Gallery
- Reviews
- Reservation Form
- Contact + Google Map
- WhatsApp CTA

## Customization Checklist
1. Replace brand name: Maison Dine
2. Replace phone number and WhatsApp link
3. Replace address and Google Map query
4. Replace menu items and prices
5. Replace images with client restaurant images
6. Adjust colors in css/style.css variables

## Run
Open `index.html` in your browser.
